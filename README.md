### ğŸ“¥ Installation

#### Method 1: Clone and Run

```bash
# 1. Clone the repository
git clone https://github.com/basem3sam/portfolio.git
cd portfolio

# 2. Start a local server (choose one):

# Option A: Python 3.x
python -m http.server 8000

# Option B: Node.js (using npx)
npx serve . -p 8000

# Option C: PHP built-in server
php -S localhost:8000

# Option D: Python 2.x (legacy)
python -m SimpleHTTPServer 8000

# 3. Open in browser
# Navigate to: http://localhost:8000
```

#### Method 2: Quick Setup (No Server)

```bash
# Simply open index.html in your browser
# Note: GitHub API features will not work without a server
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```

#### Method 3: VS Code Live Server

```bash
# 1. Install "Live Server" extension in VS Code
# 2. Right-click on index.html
# 3. Select "Open with Live Server"
# Automatically opens at http://127.0.0.1:5500
```

A: Yes! Just upload files via FTP. No special server requirements.

**Q: Can I deploy to AWS/Azure/GCP?**
A: Yes, use S3 (AWS), Static Web Apps (Azure), or Cloud Storage (GCP).

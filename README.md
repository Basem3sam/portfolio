# Basem Esam - Portfolio Website

A professional portfolio website showcasing my skills, projects, and experience as a backend developer.

## Features

- Responsive design
- GitHub API integration
- Smooth scrolling navigation
- Project showcase
- Contact information

## Setup Instructions

1. Clone or download this repository
2. Add your profile image to `assets/images/profile.jpg`
3. Add project images to `assets/images/projects/`
4. Customize content in `index.html`
5. Open `index.html` in a web browser

## Customization

- Update personal information in `index.html`
- Modify colors in `css/style.css` (CSS variables at the top)
- Add or remove projects in the projects section
- Update social media links

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- JavaScript (ES6+)
- Bootstrap 5
- Font Awesome Icons
- GitHub API
